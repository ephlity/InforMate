#01 (ФИПИ 98624A)

Задание 19

Два игрока, Петя и Ваня, играют в следующую игру. Перед игроками лежит куча камней. Игроки ходят по очереди, первый ход делает Петя. За один ход игрок может добавить в кучу один камень или увеличить количество камней в куче в два раза. Для того чтобы делать ходы, у каждого игрока есть неограниченное количество камней.

Игра завершается в тот момент, когда количество камней в куче становится не менее 69. Победителем считается игрок, сделавший последний ход, т.е. первым получивший кучу, в которой находится 69 или больше камней.

В начальный момент в куче было S камней, 1 ≤ S ≤ 68.

Будем говорить, что игрок имеет выигрышную стратегию, если он может выиграть при любых ходах противника.

Укажите минимальное значение S, при котором Петя не может выиграть за один ход, но при любом ходе Пети Ваня может выиграть своим первым ходом.

Решение

Шаг 1. Определяем диапазон значений S
Выясняем минимальное значение S для победы Вани:

  • S + 1 ≥ 69 ⇒ S ≥ 68
  • 2S ≥ 69 ⇒ S ≥ 35
  • Итого: S ∈ [1, 34]

Шаг 2. Анализируем возможные ходы
Просчитываем все возможные комбинации ходов:

  • AA — [3, 36], нам не подходит
  • AB — [2, 68], Петя выиграет
  • BA — [3, 69], Петя выиграет
  • BB — [4, 136], Петя выиграет

Шаг 3. Формируем ответ
Выходит, что при S = 34 Ваня всегда может выиграть, удвоив количество камней в куче.

Ответ: 34

Решение на Python

# Решение задания №19 (ФИПИ 98624A)

def f(s, m):
    if s >= 69: return m % 2 == 0
    if m == 0: return 0
    h = [f(s+1, m-1), f(s*2, m-1)]
    return any(h) if m % 2 != 0 else all(h)

print('19)', [s for s in range(1,69) if not f(s,1) and f(s,2)])
# Вывод: 19) [34]

За один ход игрок может добавить в кучу один или четыре камня либо увеличить количество камней в куче в два раза. У каждого игрока есть неограниченное количество камней, чтобы делать ходы.

Игра завершается в тот момент, когда количество камней в куче становится не менее 51. Победителем считается игрок, сделавший последний ход, т.е. первым получивший кучу, в которой находится 51 камень или больше.

Будем говорить, что игрок имеет выигрышную стратегию, если он может выиграть при любых ходах противника.

Укажите минимальное значение S, при котором Петя не может выиграть за один ход, но при любом ходе Пети Ваня может выиграть своим первым ходом.

Решение

Шаг 1. Определяем диапазон значений S
Обозначим ходы:

  • A: "+1" (добавить один камень)
  • B: "+4" (добавить четыре камня)
  • C: "*2" (удвоить количество камней)

Получаем уравнения для выигрышных позиций:

  • S + 1 ≥ 51 ⇒ S ≥ 50
  • S + 4 ≥ 51 ⇒ S ≥ 47
  • 2S ≥ 51 ⇒ S ≥ 26

Итого: S ∈ [1, 25]

Шаг 2. Анализируем возможные ходы

  • Если Петя сделает ход A: 26 → Ваня может сделать ход C: 52 (выигрыш)
  • Если Петя сделает ход B: 29 → Ваня может сделать ход C: 58 (выигрыш)
  • Если Петя сделает ход C: 50 → Ваня может сделать ход A: 51 (выигрыш)

Шаг 3. Проверяем минимальность
Проверим S = 24:

  • Если Петя сделает ход A: 25 → Ваня может сделать ход B: 29 → Петя может сделать ход C: 58 (выигрыш)
  • Если Петя сделает ход B: 28 → Ваня может сделать ход C: 56 (выигрыш)
  • Если Петя сделает ход C: 48 → Ваня может сделать ход B: 52 (выигрыш)

S = 24 не подходит, так как Ваня не может гарантированно выиграть.

Ответ: 25
